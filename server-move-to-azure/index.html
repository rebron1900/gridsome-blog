服务器搬家到AZURE

29 July 2014 · min read

前几天在wordpress吧里面朋友一个吧友使用的微软的azure云服务架设wordpress，网站速比我当前正在使用的国外服务器要快不知道多少倍。随心情很激动去azure的官网打算申请一个试用账号。。。但是很悲伤的发现已经停止发放激活码了，以前自己看到过azure的推广链接，那个时候还没有限制激活码的发放，不过我那个时候闲麻烦就没有申请，没想到今天也有要用到的时候。

恰巧在这个时候我使用的国外服务器速度简直慢到我不能忍受，遂直接犯傻的上淘宝找了个买家购买了一个激活码激活了azure账号。现在已经把网站搬到这边来了。azure不知道什么时候开始收费- -希望能用到回本，至少给个两个月把 。。。。。
