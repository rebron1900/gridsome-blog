Windows11下如何启用IE浏览器？新方法来了

25 July 2022 · min read

Windows11在比较新的版本里面是强制启用Edge的，常规情况下不论用什么方式都无法再打开IE浏览器，即便是在IE的安装目录里面直接点击程序图标也会强制跳转到Edge上，前些日志Betty姐似乎也遇到过这个问题。

以前使用过吾爱破解还是哪个网站的一个IE恢复工具，但是在比较新的版本里面虽然能正常安装这个补丁，但是打开IE依旧会强制跳转到Edge，给我造成了不少困扰，因为工作原因有些工具目前还只支持IE（尤其是有些银行），不能使用会很麻烦。

今天刷手机恰好收到一个公众号转载的一篇蓝点网的文章，里面讲到了两种新方法打开IE浏览器。

  • 第一种是控制面板的 Internet 选项，找到这个选项后点击对话框右上角的 ? 就会自动打开 IE 浏览器。  
  • 第二种方法是写个 vbs 脚本，脚本内容为
CreateObject("InternetExplorer.Application").Visible=true

或者你可以直接下载我写好的文件

IE启用脚本
解压后直接运行ie.vbs即可。

下次要用 IE 的时候双击这个 vbs 脚本即可，但 IE 不能固定在任务栏上，固定在任务栏上点击会跳转 Microsoft Edge。同理，在 IE 安装目录里点击 IE 的 exe 也会跳转 Microsoft Edge。  

经过我的测试是有效果的，不过IE这个东西是真的恶臭，这些银行、软件服务商真的该抛弃这个东西，连这都不弄的公司真的是有点对不住他们软件开发商这个头衔。
