图片自动缩放，防止撑破div的CSS写法

24 July 2014 · min read

wordpress插入的图片如果超过div宽度的时候如果主题没有做自动缩放就会出现图片撑破div容器的情况，就下面我修改之前喜闻乐见的样子。

如果出现这种情况我们可以通过CSS来控制：

我们需要找出文章列表最顶层容器的class，然后在css里面新建适用于所有该容易下面img标签的宽度控制样式，如下：

假设我们的顶层容器的class为context：

.context img{ MAX-WIDTH: 90%!important;HEIGHT: auto!important; }

（如果是存在于context下面的img标签的所有最大宽度为父容器的90%，高度自适应，!important语句是提高该样式的优先级别）
