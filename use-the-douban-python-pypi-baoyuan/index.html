使用豆瓣的python pypi包源

22 June 2014 · min read

easy_install默认访问的python官方网站的pypi地址，但是国内的网络。。咕~~(╯﹏╰)b你懂的，所以我们要把默认的这个访问地址更改成国内的其他包源地址就不会出现不能下载的问题了。因为我现在用的ubuntu所以就说一下ubuntu下面怎么更吧。

首先是easy_install的默认地址指定

  1. 在home下建立.pydistutils.cfg文件

  2. 添加如下语句：

[easy_install]
index_url = http://pypi.douban.com/simple/

然后就是PIP了。

  1. 在home下建立.pip文件夹

  2. 在.pip文件夹下建立pip.conf文件

  3. 在文件内添加如下代码

[global]
index-url = http://pypi.douban.com/simple/
