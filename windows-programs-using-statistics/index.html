在Windows上像智能手机那样统计软件使用情况

18 December 2021 · min read

如题，我们都知道现在的andriod、iphone等智能手机几乎都有这个app使用情况的数据统计了。打开这个功能就能很直观的看到自己的抖音、游戏或者其他摸鱼软件上浪费的生命了。

单遗憾的是windows上我一直没有这方面的意识，也没见过有这类的统计功能。哦对了，windows10有一个使用统计，但是......，我反正看不太懂。

不过昨天刷V2ex的时候偶然发现了一款软件，就像标题说的那样，可以让你在Windows上像安卓一样统计软件使用情况 。

这款软件叫Tai，github地址https://github.com/Planshit/Tai，这是一款遵循windows11 UI设计方案设计的记录你软件使用时常的程序。只要启动了 Tai ，无需再进行任何操作他就能再后台默默的记录你的摸鱼时间。

使用之前，你的电脑必须安装 .NET Framework 4.6.1 或更高的版本。然后在 releases 中下载编译好的 exe 可执行文件压缩包，下载并解压进入目录找到 Tai.exe 启动程序，成功启动后可以在状态栏看到 👻 图标。

双击图标可以进入程序主界面，右击图标显示快捷菜单。

唔，看来我一上午认真工作时间就21分钟😒
